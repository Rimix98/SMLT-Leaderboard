# План: Интеграция SMLT SMP как вкладка в основной сайт

## Контекст

Папка `smlt-smp/` содержит готовый лендинг для Minecraft SMP сервера (vanilla HTML/CSS/JS). Нужно:
1. Сделать его production-ready (мерж в основной код)
2. Добавить вкладку "SMLT SMP" в основной Vue 3 сайт

## Подход

Создать Vue-компонент `SmpPage.vue`, который воспроизводит содержимое лендинга в стиле основного сайта, используя `AppShell` + CSS-переменные темы. Оригинальная папка `smlt-smp/` остаётся как standalone-версия.

## Изменения

### 1. Новый компонент: `Frontend/src/components/SmpPage.vue`

Страница с секциями:
- **Hero** — заголовок "SMLT SMP", описание, кнопки "Начать играть" / "Открыть карту", IP-адрес с копированием
- **Статус сервера** — карточка со статусом, игроками, версией, IP (данные захардкожены как в оригинале)
- **О сервере** — три карточки преимуществ (Выживание, Сообщество, События)
- **Карта** — блок с описанием + стилизованная CSS-карта (без iframe, он закомментирован в оригинале)
- **Правила** — 6 правил сервера
- **Как начать играть** — 5-шаговая инструкция + карточка с IP
- **Футер** — копирайт, ссылки (Discord, карта, правила)

Особенности реализации:
- Использует `AppShell` с `#brand` слотом (иконка + "SMLT SMP")
- Все стили пишутся в `<style scoped>` с использованием CSS-переменных основного сайта (`--color-*`, `--spacing-*`, `--border-radius-*`)
- Копирование IP через `navigator.clipboard` + fallback (как в оригинале)
- Анимации `reveal` через `IntersectionObserver` (как в оригинале)
- Статус сервера: захардкоженные данные (как в оригинале), с возможностью замены на API
- Hero-фон: градиент без изображения (оригинальный `hero.webp` не копируем в `Frontend/public/`, чтобы не раздувать репозиторий)

### 2. Роутинг: `Frontend/src/router.ts`

Добавить маршрут:
```ts
{ path: '/smp', name: 'smp', component: () => import('./components/SmpPage.vue'), meta: { title: 'SMLT SMP — Minecraft-сервер', bodyClass: '' } }
```

### 3. Навигация: `Frontend/src/components/AppShell.vue`

Добавить ссылку в `<nav class="header-nav">`:
```html
<router-link to="/smp" class="nav-link" :class="{ active: currentPage === 'smp' }">SMP</router-link>
```
